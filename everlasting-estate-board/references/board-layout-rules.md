# Board Layout Rules

## Default canvas

- Ultra-wide landscape board
- Recommended: 2400 x 1350, 3200 x 1800, or 3840 x 2160
- Print-friendly option: 24 x 36 inch landscape

## Section grid

1. Hero exterior: 30% height
2. Plan/map band: 28% height
3. Interior vignette strip: 12% height
4. Lifestyle vignette strip: 12% height
5. SCIP/resilience strip: 10% height
6. Footer brand line: 8% height

## Reference-style layout

Top:
- Large photorealistic exterior estate image
- Estate title top left
- Legacy statement top right

Middle:
- Main floor plan panel
- Upper floor panel
- Wellness/lower level panel
- Estate site plan panel

Lower:
- Life Lived Beautifully image strip
- Lifestyle image strip
- SCIP advantage icons and blurbs
- Footer signature line

## Visual rules

- Use generous margins.
- Keep gutters thin and elegant.
- Use pale paper texture or warm ivory base.
- Keep captions small but legible.
- Use small caps section labels.
- Avoid clutter and excessive labels.
- Prioritize hierarchy over filling every inch.
