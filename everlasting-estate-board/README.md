# Everlasting Estate Board Claude Skill

A custom Claude/Claude Code skill for generating luxury architectural dream-home concept boards from family intake forms.

## Install locally

Copy the `everlasting-estate-board` folder into your Claude skills directory or project skill/plugin structure, depending on your Claude Code setup.

## Use

Ask Claude Code:

> Use the everlasting-estate-board skill to turn this intake form into a luxury dream-home concept board prompt and HTML layout direction.

## Output

The skill produces:

- Estate title and subtitle
- Legacy statement
- Floor-plan fantasy zones
- Site-plan zones
- Interior and lifestyle vignette captions
- SCIP/resilience modules
- Final image prompt
- Optional HTML/CSS layout direction

## Note

This produces conceptual design direction only. It does not create construction-ready architecture, engineering, or permit drawings.
